## 第1章 総則 

<|ref|>text<|/ref|><|det|>[[108, 203, 206, 220]]<|/det|>
〔目 的〕 

<|ref|>text<|/ref|><|det|>[[88, 224, 920, 285]]<|/det|>
第1条 株式会社ソミック石川 下野部工場 (以下「当事業場」という。) における電気工作物の工事、維持及び運用に関する保安を確保するため、電気事業法第42条第1項の規定に基づき、この規程を定める。 

<|ref|>text<|/ref|><|det|>[[108, 311, 206, 328]]<|/det|>
〔効 力〕 

<|ref|>text<|/ref|><|det|>[[88, 332, 744, 350]]<|/det|>
第2条 当事業場の経営者及び従業者は、電気関係法令及びこの規程を遵守する。 

<|ref|>text<|/ref|><|det|>[[108, 375, 225, 392]]<|/det|>
〔細則の制定〕 

<|ref|>text<|/ref|><|det|>[[88, 396, 781, 414]]<|/det|>
第3条 この規程を実施するため、必要を認められる場合には、別に細則を制定する。 

<|ref|>text<|/ref|><|det|>[[108, 440, 243, 457]]<|/det|>
〔規程等の改正〕 

<|ref|>text<|/ref|><|det|>[[88, 461, 917, 500]]<|/det|>
第4条 この規程又は前条に定めた細則の制定又は改正にあたっては、電気主任技術者の参画のもとに立案し、これを決定する。 

<|ref|>sub_title<|/ref|><|det|>[[340, 543, 658, 564]]<|/det|>
## 第2章 保安業務の運営体制 

<|ref|>text<|/ref|><|det|>[[108, 590, 263, 607]]<|/det|>
〔保安業務の監督〕 

<|ref|>text<|/ref|><|det|>[[88, 611, 917, 671]]<|/det|>
第5条 電気工作物の工事、維持及び運用に関する保安業務は、工場長 (以下「総括管理者」という。) が統括管理し、電気主任技術者を別図第1のように配置して、その監督にあたらせる。 

<|ref|>text<|/ref|><|det|>[[88, 697, 650, 715]]<|/det|>
第6条 電気主任技術者の保安監督の職務は、次の事項について行う。 

<|ref|>text<|/ref|><|det|>[[135, 719, 664, 737]]<|/det|>
(イ) 電気工作物にかかる従業者に対する保安教育に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 741, 448, 758]]<|/det|>
(ロ) 電気工作物の工事に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 762, 448, 780]]<|/det|>
(ハ) 電気工作物の保守に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 784, 482, 801]]<|/det|>
(ニ) 電気工作物の運転操作に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 805, 370, 822]]<|/det|>
(ホ) 災害対策に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 827, 427, 844]]<|/det|>
(ヘ) 保安業務の記録に関すること。 

<|ref|>text<|/ref|><|det|>[[135, 848, 515, 865]]<|/det|>
(ト) 保安用機材及び書類の整備に関すること。 

<|ref|>text<|/ref|><|det|>[[123, 869, 915, 887]]<|/det|>
2 電気主任技術者は、電気工作物の工事、維持及び運用に関する保安の監督の職務を誠実に行う。
